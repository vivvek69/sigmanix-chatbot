# Replit Quick Start - 3 Minutes!

## Super Fast Setup

### Step 1: Go to Replit (30 seconds)
https://replit.com → Sign up (if needed)

### Step 2: Import Project (1 minute)
1. Click **+ Create**
2. Click **Import from GitHub**
3. Paste: `https://github.com/vivvek69/sigmanix-chatbot`
4. Click **Import**

### Step 3: Add Secret (30 seconds)
1. Click **Secrets** (lock icon)
2. Click **New Secret**
3. Name: `GROQ_API_KEY`
4. Value: Your Groq API key
5. Click **Add Secret**

### Step 4: Run (30 seconds)
1. Click **RUN** button
2. Wait for: `Listening on 0.0.0.0:5000`
3. Click **Web** tab
4. Your URL is ready!

## Done! 🎉

Your chatbot is LIVE at: `https://yourname-sigmanix-chatbot.replit.dev`

**Test it now!**

## Get Groq API Key

1. Go to https://console.groq.com/
2. Sign up (free)
3. Create API key
4. Copy and paste in Replit Secret
